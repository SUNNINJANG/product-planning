# 프로젝트 진행현황

| 제품명 | 단계 | 상태 | 비고 |
|--------|------|------|------|
| 공간 탈취제 (하우스보이) | 기획 중 | 🟢 진행 | 오브제형 탈취제, 리필 모델 |

## 상태 범례

- 🟢 정상 진행

- 🔴 문제 발생
- ⚪ 대기
- ⛔ Drop
