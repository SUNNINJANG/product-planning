## 사용 시나리오

1. **축구·태권도 다녀온 아이 유니폼** — 유아용 세제를 쓰는데 통에 따로 세탁하기 부담스러운 부모. 유니폼에 뿌리고 그냥 돌리면 땀 냄새 없이 깔끔해짐.
2. **땀 많은 남편 옷** — 매번 분리 세탁하거나 냄새 확인하며 특수 세제로 따로 돌리는 번거로움. 남편 옷에 뿌려두고 가족 빨래 한 번에 같이 해결.
3. **필라테스·요가 하는 직장인 여성** — 레깅스나 스포츠 브라는 땀냄새 때문에 세탁 후에도 다시 입기 찜찜한 경우. 세탁 전에 뿌려두면 전용 세제 없이도 일반 세탁으로 냄새까지 잡힘.
4. **매번 소량 분리 세탁이 비효율적인 사람** — 운동복에 뿌리고 한 번에 같이 돌리는 선택지 제공.

## 기술 / 원료 리서치

### ① 세제와의 충돌 방지 → 양쪽성 또는 비이온성 항균 원료

| 원료명 | 특징 |
|--------|------|
| Lauryl Betaine (라우릴 베타인) | 양쪽성 계면활성제. 음이온 세제와 충돌 없음 |
| PHMB (Polyhexamethylene Biguanide) | 비이온성에 가까운 항균제. 세제 존재 하에서도 효능 유지 |
| Zinc Pyrithione | 비이온성 항균 성분. 세탁 중에서도 안정적 |

### ② 헹굼에서 씻겨나가는 문제 → 섬유 결합형 원료

| 원료명 | 특징 |
|--------|------|
| Si-QAC (Silicone Quaternary Ammonium Compound) | 실리콘 기반, 섬유 흡착력 매우 강함. BIORESTORE가 쓰는 성분 |
| Chitosan (키토산) | 천연 유래. 섬유에 이온 결합하여 헹굼 후에도 잔류 |
| HeiQ Fresh | 스위스 HeiQ社 B2B 원료. 헹굼 내성 검증된 항균 소취 성분 |

### ③ 합성섬유 흡착 → 침투 보조제 조합

| 원료명 | 역할 |
|--------|------|
| Cyclodextrin (사이클로덱스트린) | 냄새 분자를 포집하는 캡슐 구조. 소재 상관없이 냄새 원인 물질 자체를 잡음 |
| Ethanol (에탄올) | 합성섬유 표면 장력을 낮춰 성분 침투 도움 |
| Sanitized AG (스위스) | 합성섬유 특화 항균 원료 B2B 공급 전문 원료사 |

### 해결 가능 여부 정리

| 영역 | 상태 | 방법 |
|------|------|------|
| 항균·소취 핵심 성분 | ✅ 해결 가능 | 기존 원료 조합 |
| 세제 충돌 방지 | ✅ 해결 가능 | 원료 선택 |
| 합성섬유 침투 | ✅ 해결 가능 | 사이클로덱스트린 + 에탄올 |
| 비율·포뮬러 최적화 | ⚠️ 협업 필요 | ODM 공장과 협업 |
| 효능 테스트·인증 | ⚠️ 의뢰 필요 | 시험기관 |
