## 브랜드 컨셉

더티클레이의 패키지 디자인은 **마트 식료품 코너**에서 영감을 받습니다.

기존 더티클레이 버터가 "버터 같은" 패키지로 리뉴얼되면서, 식료품이라는 세계관 안에서 자연스럽게 시리즈를 확장할 수 있는 구조입니다.

> 동네 식료품점의 냉장 코너를 그대로 매대로 옮겨온 컨셉.
> 나중에 AI로 캠페인 영상을 만들어도 재밌을 것.

### 왜 유제품 레퍼런스인가

- **질감의 일치**: 크림, 우유, 요거트의 부드럽고 하얀 질감 → 폼 건 거품과 시각적으로 바로 연결
- **청결 이미지**: 유제품의 순백 이미지 = "깨끗함". 올리브오일처럼 "기름=더러움" 역연상 없음
- **시리즈 확장성**: 유제품 코너처럼 다양한 제품을 하나의 세계관에서 진열 가능

> 아래는 확정 라인업이 아님. 폼 건과 유제품이 왜 컨셉적으로 잘 맞는지에 대한 레퍼런스 정리.

<details>
<summary><b>휘핑크림 — 폼 질감과 가장 직관적으로 연결</b></summary>

| 항목 | 내용 |
|------|------|
| 분사 방식 | 에어로졸 분사 = 폼 건의 거품 분사와 동일한 사용 경험 |
| 형태 | 세로형 캔/보틀 형태가 폼 건 용기와 일치 |
| 질감 연상 | 하얀 크림 거품 = 폼 건에서 나오는 쫀득한 폼 |
| 이미지 | 크림의 하얀색 = 깨끗함, 순수함과 자연스럽게 연결 |
| 디자인 참고 | Reddi-Wip 스타일 레트로 라벨, 빈티지 유제품 광고 |

</details>

<details>
<summary><b>우유 — 청결 이미지의 원형</b></summary>

| 항목 | 내용 |
|------|------|
| 이미지 | 우유 = 순백 = 깨끗함의 가장 보편적 상징 |
| 형태 | 밀크보틀 실루엣을 라벨 디자인에 활용 가능 |
| 컬러 | 빨강+파랑 클래식 유업 컬러가 매대에서 시선 끌기 좋음 |
| 세계관 | 유제품의 "기본"이므로 시리즈 정체성을 잡아주는 역할 |
| 디자인 참고 | 빈티지 유리 밀크보틀 라벨, 레트로 배달 우유 패키지 |

</details>

<details>
<summary><b>요거트 — 질감의 밀착감 + 확장성</b></summary>

| 항목 | 내용 |
|------|------|
| 질감 | 쫀득하게 늘어나는 요거트 점성 = 벽면에 밀착하는 폼 건 거품 |
| 확장성 | 과일맛 요거트 시리즈 → 향 변형 라인업으로 확장 가능 |
| 톤 | 파스텔 톤 + 귀여운 일러스트로 친근한 이미지 |
| 프리미엄 | 그릭요거트 감성으로 프리미엄 라인 분리도 가능 |
| 디자인 참고 | 수제 요거트 브랜드 패키지, 과일 일러스트 라벨 |

</details>

## 브랜드 스토리 방향

<details>
<summary><b>세계관 & 톤앤매너</b></summary>

### 세계관

더티클레이 버터에서 시작된 **"식료품 같은 청소용품"** 세계관.
동네 식료품점 냉장 코너에 나란히 놓인 식품들처럼, 더티클레이의 청소 제품들이 하나의 식료품 브랜드 시리즈로 보이는 것이 목표.

### 핵심 메시지

- **겉보기**: "어, 이거 먹는 거 아니야?" — 식료품처럼 보이는 패키지의 의외성
- **사용할 때**: "거품이 진짜 크림 같다" — 폼 건 거품과 유제품 질감의 감각적 연결
- **시리즈로 모았을 때**: 수집욕을 자극하는 라인업 구성

### 톤 앤 매너

- 유쾌하지만 촌스럽지 않은 키치
- 빈티지 감성이지만 깔끔하게 정제된 레트로
- 식료품의 친근함 + 청소용품의 기능적 신뢰감 균형

### 바이럴 포인트

- "이게 세제라고?" 반전 → SNS 언박싱/리뷰 콘텐츠
- 식료품 코너처럼 진열한 인증샷 → 사용자 생성 콘텐츠(UGC)
- AI 캠페인 영상 — 마트 식료품 코너에 청소용품이 진열된 영상 등

</details>
